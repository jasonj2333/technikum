Orkiestra

Wiele pojęć z rzeczywistego świata jesteśmy w stanie przenieść bezpośrednio na język obiektowo-orientowany. Rozważmy na przykład kilka instrumentów muzycznych, np. gitarę, flet i puzon. Każdy z nich wygląda i działa inaczej, ale wszytkie posiadają jedną istotną cechę - można na nich grać! Powiemy zatem, że:

  • Gitara jest Instrumentem i można na niej grać.
  • Flet jest Instrumentem i można na nim grać.
  • Puzon jest Instrumentem i można na nim grać.

Najczęściej słowo jest określa nam relację między typami danych, która odpowiada dziedziczeniu. Widzimy więc, że w tym przypadku Instrument będzie pełnił rolę klasy bazowej... ale czy na pewno klasy? Czy możemy powiedzieć, czym jest i w jaki sposób zachowuje się po prostu "instrument"? Wiemy jedynie, że może grać, ale nie wiemy w jaki sposób. W takiej sytuacji powinniśmy posłużyć się interfejsem. W C++ interfejsy realizujemy jako klasy abstrakcyjne zawierające tylko metody czysto wirtualne, ewentualnie definicje metod statycznych.

Napisz deklaracje dla wszystkich wymaganych typów i przetestuj ich działanie:

  1. Stwórz klasy Guitar, Flute i Trombone które definiują odpowiednią metodę play().
  2. W funkcji main() zagraj na każdym z instrumentów. Czy możesz zagrać na instrumentach w pętli?
  3. Stwórz klasę bazową Instrument i zadeklaruj w nim metodę play(). Zaimplementuj dziedziczenie w stworzonych wcześniej klasach.
  4. Spróbuj teraz zagrać na swoich instrumentach w pętli, tworząc z nich wcześniej odpowiednią kolekcję (tablicę). Co obserwsujesz? Jaką rolę w realizacji zadania pełni polimorfizm?
  5. Zmień klasę bazową w klasę abstrakcyjną (Interfejs) zawierającą tylko metodę czysto wirtualną play().
  6. Stwórz klasę Orchestra, która:
    • również dziedziczy po Instrument
    • posiada pole prywatne name, typu string oraz publiczną metodę setName() ustawiającą pole name
    • posiada pole prywatne instruments, który jest listą (vectorem) obiektów typu Instrument
    • posiada metodę addInstrument(Instrument), która dodaje do orkiestry nowy instrument
    • definiuje metodę play(), w której gra na wszystkich instrumentach - wołając w pętli na każdym z nich play().
  7. W funkcji main() stwórz instancje orkiestry i dodaj do niej kilka różnych instrumentów. Zagraj na orkiestrze.

Przykład zastosowania interfejsu w Javie:

public interface Vehicle {
        void drive();
    }
     
    public class Car implements Vehicle {
        public void drive() {
           // tu odpalamy silnik i jedziemy
        }
    }
     
    public class Bike implements Vehicle {
        public void drive() {
           // tu pedałujemy!
        }
    }
