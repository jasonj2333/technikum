Obiektowy Model Dokumentu (DOM - Document Object Model)

DOM to obiektowy model dokumentu HTML, który pozwala JS:

  • odczytywać elementy strony,
  • modyfikować ich treść i style oraz atrybuty,
  • reagować na zdarzenia(kliknięcia, wpisywanie tekstu itp.).

Dzięki DOM strona może się zmienić bez przeładowywania.
